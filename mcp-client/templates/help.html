PEC MCP · Ajuda

Manual introdutório para uso pela interface.

Voltar ao Console

O PEC MCP é uma ferramenta de Inteligência Clínica e Gestão integrada ao e-SUS PEC. Ela permite consultar dados do prontuário com linguagem natural, mantendo tudo anonimizado e seguro, sem necessidade de conhecimento técnico.

1) Chat com IA

Digite perguntas claras e objetivas. O chat entende melhor quando você informa pelo menos um critério (ID, unidade, sexo, idade, condição ou CID/CIAP).

2) Auto-tools (atalhos)

Clique nos atalhos da interface ou digite / no chat. Os auto-tools executam consultas prontas com poucos cliques.

Exemplos prontos para o chat

Copie e cole, trocando apenas os dados do seu contexto.

Assistência clínica

  • “Mostre um resumo clínico do paciente de ID 123.”
  • “Quais condições ativas esse paciente possui?”
  • “Quando foi o último atendimento registrado?”
  • “Liste até 3 atendimentos recentes desse paciente.”

Busca ativa

  • “Liste as gestantes sem consulta nos últimos 2 meses”
  • “Quantos pacientes com diabetes nao tiveram consultas nos últimos 6 meses?”
  • “Faça uma lista dos 20 primeiros dessa lista”
  • “Temos algum registro de tuberculose recente?”

Gestão e planejamento

  • “Quantos pacientes com hipertensão existem nesta unidade?”
  • “Quantos diabéticos acima de 50 anos temos?”
  • “Compare a quantidade de hipertensos entre as unidades.”
  • “Qual a carga de doenças crônicas na unidade?”

Dicas rápidas

  • Sempre informe algum critério (idade, sexo, unidade, condição, CID ou ID).
  • Frases curtas funcionam melhor.
  • Evite perguntas abertas como “me mostre tudo”.
  • Use o filtro de Unidade no console para refinar os resultados.

Auto-tools (atalhos da interface)

Para abrir o menu, digite / no chat.

/paciente-id

Use quando já tem o ID interno do paciente. A tela abre abas com dados básicos (anonimizados), histórico de atendimentos (S/O/A/P) e condições de saúde.

/saude-360-c3

Ideal para acompanhar gestantes e o indicador C3 do Previne Brasil. Você pode filtrar por período ou unidade e ver a lista de pacientes em déficit.

Auto-tools marcadas como “Em breve” aparecem desabilitadas no menu.

Quando usar chat ou auto-tools?

Use o chat quando:

  • Você quer explorar um cenário ou tirar dúvidas.
  • Precisa de respostas em linguagem natural.
  • Quer ajustar filtros rapidamente.

Use auto-tools quando:

  • Já sabe exatamente o que precisa.
  • Busca rapidez e padronização.
  • Quer repetir a mesma consulta várias vezes.

Segurança e privacidade

  • O sistema nunca mostra nomes completos ou documentos.
  • Os dados aparecem anonimizados (iniciais, sexo, datas e condições).
  • O PEC MCP apenas lê o prontuário, não altera informações.

Resumo: use o chat para perguntar, use auto-tools para agir rápido.
